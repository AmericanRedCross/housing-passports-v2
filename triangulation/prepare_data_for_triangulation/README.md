# housing passports data scripts

This is a bunch of scripts to process the files for Housing Passport v2.

## Build Docker image

Run the follow comand to build docker image

```shell

docker-compose build   

```


## Scripts

The script are compiled in the `index.sh` file

```shell
bash index.sh

```

prepare files for db_compilation, run `post_processing.sh`

```shell
bash post_processing.sh

```
